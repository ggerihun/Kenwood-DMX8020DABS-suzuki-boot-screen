# Kenwood-DMX8020DABS-suzuki-boot-screen
suzuki swift boot screen for Kenwood DMX8020DABS 


On PC:
Place files on fat32 usb and plug it into the headunit.

On Headunit:
menu -> standby -> (topleft tap, topleft tap, bottomleft tap, bottomleft tap, topleft tap, bottomleft tap) -> customize menu -> opening customize
